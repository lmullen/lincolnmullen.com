• Omeka Client — a Ruby gem that provides a REST client for the Omeka API, to access Omeka sites programmatically. (GitHub | RubyGems)
  • Catalog Search Plugin — a plugin that links Omeka to searches in catalogs such as Archive Grid and the Library of Congress. (GitHub | Omeka Plugins)
  • Jekyll E-book — a Ruby gem to create EPUB e-books from Jekyll posts and pages using Pandoc.
  • Bibkeys — a Ruby gem to list all the citation keys in a BibTeX file.
  • Honor Thy Contributors Plugin — a plugin to give credit to your contributors on an Omeka site.
  • Add Item Plugin — a simple Omeka plugin that adds an "add item" link to the admin bars.
  • History Bibliography — a BibTeX database of citations in religious history and American history.
  • Academic Article LaTeX Class — a LaTeX class suitable for academic articles in the humanities.
  • Academic Paper LaTeX Class — a LaTeX class suitable for classroom papers.
  • Dotfiles — my Unix/Linux/Mac OS X configuration files, mostly for ZSH, Vim, Git, Ruby, and R.
