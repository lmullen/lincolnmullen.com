Reproducible Research

Reproducible research is

  1. documented,
  2. version controlled,
  3. automated with a build script,
  4. written as a literate program,
  5. with all dependencies managed.

My attempt at explaining reproducible research.

More Unix

Why Vim keeps popping up:

  • The $EDITOR shell variable.
  • Enough Vim to make you dangerous: i to insert; ESC to get back to to command mode; : to execute a command; :w to write; :q to quit; :wq to write and quit; hjkl to move around.
  • Try export EDITOR='subl -w' in your ~/.bash_profile or ~/.zshrc.
  • Another option on Macs is export EDITOR='open -a TextEdit'.

Executable files:

  • Execute a script: Rscript script.R or ruby script.rb or node script.js or python script.py or bash script.sh.
  • The hashbang or shebang: #!.
  • Execute a script without specifying a command by putting a hashbang at the first line of the file: #!/usr/bin/node.
  • Except the correct binary for the language might not be in /usr/bin/, so use #!/usr/bin/env node.
  • Make a file executable with chmod +x script.js.
  • Use ls -l to check permissions. Explanation of permissions.
